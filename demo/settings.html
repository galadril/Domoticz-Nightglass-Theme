How it works

  1. Go to Settings → Nightglass tab in Domoticz
  2. Find the Device Icon Overrides section and click Manage
  3. Search for a device in the list (or use a Quick Preset chip)
  4. Click + next to any device, then pick an icon from the searchable grid
  5. Set custom On and Off colors with the color pickers
  6. Click Save Overrides — overrides persist across browsers via Domoticz user variables
